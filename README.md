🚀 Java Network Monitoring Tool
📌 Overview
A multi-threaded Java application that monitors network latency of multiple hosts in real time and logs performance data.

✨ Features
Real-time latency monitoring
Multi-threading support
File logging (CSV format)
Alert system for high latency
Scalable to multiple hosts
🛠 Tech Stack
Java
Multithreading
Socket Networking
▶️ How to Run
Compile: javac Main.java
Run: java Main
📊 Output Example
google.com -> 15 ms facebook.com -> 14 ms
